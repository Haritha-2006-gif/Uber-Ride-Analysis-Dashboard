# 🚖 Uber Ride Analysis Dashboard

## 📌 Project Description

The Uber Ride Analysis Dashboard is an interactive Power BI project developed to analyze ride booking data and provide actionable business insights. The dashboard helps monitor bookings, revenue, ride cancellations, customer preferences, and driver performance through dynamic visualizations and KPI tracking.

This project demonstrates the use of Power BI, Power Query, and DAX to transform raw data into meaningful business intelligence.



## 🎯 Business Problem Statement

Ride-sharing companies generate large amounts of data every day. Without proper analysis, it becomes difficult to identify revenue trends, customer behavior, vehicle performance, and operational issues.

This dashboard provides a centralized solution for tracking key business metrics and supporting data-driven decision-making.



## 🛠️ Tech Stack

* **Power BI** – Dashboard Development
* **Power Query** – Data Cleaning & Transformation
* **DAX** – Calculated Measures and KPIs
* **Microsoft Excel** – Data Source
* **GitHub** – Project Documentation and Version Control



## ✨ Features

### 🏠 Home Dashboard

* Total Bookings
* Total Revenue
* Average Ride Distance
* Average Driver Rating
* Vehicle Type Analysis
* Payment Method Analysis

### ❌ Cancellation Analysis

* Customer Cancellation Reasons
* Driver Cancellation Reasons
* Cancellation Trends
* Cancellation Distribution

### 📈 Performance Analysis

* Driver Performance Metrics
* Ride Distance Analysis
* Revenue Trends
* Weekday Performance Analysis
* Operational KPI Monitoring



## ❓ Key Business Questions

* How many rides were booked?
* What is the total revenue generated?
* Which vehicle category is most popular?
* Which payment method is preferred by customers?
* What are the major reasons for ride cancellations?
* How do driver ratings impact performance?
* Which days generate the highest ride demand?

## 📊 Key Insights

* UPI is the most preferred payment method among customers.
* Auto and Go Mini categories contribute significantly to total bookings.
* Driver ratings remain consistently high.
* Revenue trends vary across different months.
* Cancellation analysis helps identify operational improvement opportunities.



## ✅ Conclusion

The Uber Ride Analysis Dashboard provides a comprehensive overview of ride-sharing operations by combining booking, revenue, customer, and performance metrics into a single reporting solution.

This project showcases practical skills in data cleaning, data modeling, DAX, dashboard design, and business analytics while delivering meaningful insights for decision-making.
